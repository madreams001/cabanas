⚙️ Configuración inicial necesaria

Esta app necesita conectarse a Google Sheets. Seguí estos pasos una sola vez:

  1. Abrí Google Sheets y creá una nueva planilla. Copiá el ID de la URL.
  2. Abrí script.google.com → Nuevo proyecto.
  3. Pegá el contenido del archivo Codigo_AppScript.gs y reemplazá XXXXXXXXXX por el ID de tu Sheet.
  4. Guardá → Implementar → Nueva implementación → Tipo: Aplicación web → Acceso: Anyone.
  5. Copiá la URL generada y pegala en este archivo HTML reemplazando PEGAR_AQUI_LA_URL_DE_APPS_SCRIPT.
⚠️ Hasta que configures la URL, la app funciona en modo demo (los datos no se guardan en Google Sheets).
