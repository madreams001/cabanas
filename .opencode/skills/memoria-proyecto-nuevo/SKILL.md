## Instrucciones para usar este template

1. Copiar este directorio: `.opencode/skills/memoria-proyecto-nuevo/`
2. Renombrarlo: `.opencode/skills/memoria-[nombre-del-proyecto]/`
3. Renombrar el archivo `SKILL.md` con el nombre del proyecto
4. Actualizar el `name:` y `description:` en el frontmatter
5. Rellenar cada sección con la info del proyecto nuevo
6. Commit a git
